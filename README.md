### nodejs-mysql-crud

##### Step 1: Install docker.

##### Step 2: create a .env file in the project root with your passkeys for:

```
DB_HOST=
DB_USER=
DB_PASSWORD=
DB_PORT=
DB_NAME=
DB_ROOT_PASSWORD=
```

**After this, run this command:**

"docker compose up -d"

Access: http://localhost:3000/
